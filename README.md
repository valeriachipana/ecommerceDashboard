# Dashboard de KPIs para Ecommerce (Power BI)

## Herramientas utilizadas
- **Power BI**: visualización, modelado y diseño del dashboard
- **DAX**: creación de medidas, KPIs y cálculos personalizados
- **Power Query (M)**: limpieza, transformación y preparación de datos
- **Excel**: exploración inicial y validación de datos
- **PowerPoint**: apoyo en diseño visual y paleta de colores

## Descripción
Este proyecto consiste en el desarrollo de un **dashboard interactivo en Power BI** para un ecommerce, orientado al **seguimiento y análisis de KPIs clave** de las áreas de marketing, logística y administración.

Las etapas del proyecto incluyen: ETL, modelado de datos, uso de DAX, filtros dinámicos, storytelling visual y aplicación de principios preatentivos, de color y Gestalt aplicados a la visualización de datos.

## Objetivo del proyecto
Diseñar un tablero dinámico, escalable y reutilizable que permita:

- Monitorear KPIs clave del negocio
- Comparar resultados entre períodos
- Facilitar el análisis exploratorio mediante filtros interactivos y análisis de cuellos de botella
- Apoyar la toma de decisiones tácticas y estratégicas

El dashboard está preparado para actualizarse fácilmente con nuevos datos sin necesidad de rediseñar la estructura.


## Usuarios finales
El tablero está pensado para distintos perfiles dentro de la organización:

- **Marketing:** análisis de comportamiento de compra, métodos de pago, horarios y categorías más vendidas
- **Logística:** seguimiento de entregas, demoras, cuellos de botella y desempeño por ciudad
- **Administración:** monitoreo general del desempeño del negocio y evolución de KPIs

Nivel de aplicación: **táctico–estratégico**.

## Enfoque de diseño y visualización
El diseño del dashboard se apoya en buenas prácticas de visualización y UX:

- Uso de **atributos preatentivos** (color, tamaño, posición) para resaltar información clave
- Aplicación de **principios Gestalt** para facilitar la lectura y agrupación visual
- Psicología del color alineada a la identidad visual del ecommerce
- Indicadores con **comparación vs período anterior**, utilizando colores y variaciones porcentuales para una interpretación rápida
- Inclusión de **ayudas visuales y texto contextual** para facilitar el uso por personas no técnicas

El objetivo es que cualquier usuario pueda interpretar el tablero sin necesidad de explicaciones externas.


## Funcionalidades principales del dashboard
- KPIs dinámicos con variación respecto al período anterior
- Segmentaciones por:
  - Año y mes
  - Región / ciudad
- Comparación temporal de métricas clave
- Visualizaciones interactivas que responden al filtrado del usuario
- Navegación clara entre vistas (visión general, marketing, logística, administración)
